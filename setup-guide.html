วิธีเชื่อมต่อแต่ละแพลตฟอร์ม
ทีละขั้นตอน

เลือกแพลตฟอร์มที่คุณต้องการเชื่อมต่อ แล้วทำตามขั้นตอนด้านล่าง มีทั้งภาพประกอบและวิดีโอสอน — ตั้งค่าเองได้ ไม่ต้องง้อช่าง

LINE Official Account

เชื่อมต่อ LINE OA ของคุณกับ NextGenBot เพื่อตั้งค่า Auto-Reply, Flex Message และแคมเปญการตลาด

1 สมัคร LINE Official Account

ไปที่ https://manager.line.biz แล้วสมัครบัญชี LINE OA ฟรี ถ้ามีอยู่แล้วข้ามขั้นตอนนี้

2 สร้าง Messaging API Channel

เข้า LINE Developers (https://developers.line.biz) → สร้าง Provider → สร้าง Messaging API Channel และรับ Channel Access Token

3 ตั้งค่า Webhook

ใน LINE Developers ใส่ Webhook URL ที่ได้จาก NextGenBot: https://api.nextgenbot.com/webhook/line และกด Verify

4 เชื่อมต่อกับ NextGenBot

ในหน้า Settings ของ NextGenBot ใส่ Channel Access Token และ Channel Secret ที่ได้จาก LINE Developers แล้วกด Save — เสร็จ!

💡 Tips: ควรเปิด Use Webhook และ Auto-Reply ใน LINE OA Manager ถึงจะทำงานได้สมบูรณ์

Facebook Messenger

เชื่อมต่อเพจ Facebook ของคุณเพื่อรับ-ส่งข้อความกับลูกค้าผ่าน NextGenBot

1 สร้าง Meta App

ไปที่ https://developers.facebook.com → My Apps → สร้างแอปใหม่ เลือก Business

2 เพิ่ม Messenger Product

ใน Dashboard กด Add Product → เลือก Messenger → ตั้งค่าให้เชื่อมต่อกับเพจของคุณ

3 สร้าง Page Access Token

เลือกเพจ → Generate Token → คัดลอก Page Access Token มาเก็บไว้

4 ตั้งค่า Webhook + เชื่อมต่อ

ใส่ Webhook URL: https://api.nextgenbot.com/webhook/messenger → ใส่ Verify Token → Subscribe to messages → นำ Token ไปใส่ใน NextGenBot

💡 Tips: กรุณาเปิด Required Permissions: pages_messaging, pages_manage_metadata ถึงจะใช้งานได้ครบ

WhatsApp Business

เชื่อมต่อ WhatsApp Business API เพื่อตอบแชทและส่งแคมเปญผ่าน WhatsApp

1 ลงทะเบียน WhatsApp Business API

ผ่าน Meta Business Manager → WhatsApp Account → ขอหมายเลขโทรศัพท์และยืนยัน

2 สร้าง Access Token

ใน Meta for Developers → WhatsApp → สร้าง Permanent Token สำหรับใช้งาน

3 ตั้งค่า Webhook

ใส่ Webhook URL: https://api.nextgenbot.com/webhook/whatsapp → ตั้งค่า Verify Token

4 เชื่อมต่อกับ NextGenBot

เข้า Settings ใน NextGenBot → WhatsApp → ใส่ Phone ID + Token + Webhook Secret ที่ได้ — เสร็จสิ้น

💡 Tips: WhatsApp Business API มีค่าบริการตามการสนทนา (conversation-based pricing) — เช็กอัตราก่อนเริ่มใช้

Telegram Bot

สร้าง Telegram Bot ด้วย BotFather และเชื่อมต่อกับ NextGenBot

1 พิมพ์ @BotFather ใน Telegram

ค้นหา BotFather ใน Telegram แล้วพิมพ์ /newbot เพื่อสร้างบอทใหม่

2 รับ Token

ตั้งชื่อบอทและ username → BotFather จะส่ง HTTP API Token มาให้ — คัดลอกเก็บไว้

3 ตั้งค่า Webhook

ใช้ URL: https://api.telegram.org/bot<TOKEN>/setWebhook?url=https://api.nextgenbot.com/webhook/telegram

4 เชื่อมต่อกับ NextGenBot

นำ Telegram Bot Token ไปใส่ใน NextGenBot Settings แล้วกด Save — เสร็จ!

💡 Tips: ปิด Privacy Mode ใน BotFather ด้วย /setprivacy ถ้าต้องการให้บอทอ่านทุกข้อความใน group

TikTok

เชื่อมต่อ TikTok Business Account เพื่อจัดการข้อความและคอมเมนต์

1 สร้าง TikTok Business Account

สมัคร TikTok Business Account ที่ https://www.tiktok.com/business

2 สร้าง TikTok App

ไปที่ TikTok Developers → สร้าง App → เลือก permissions: user.info.basic, video.upload, message.send

3 รับ Access Token

ทำ OAuth 2.0 เพื่อขอ Access Token และ Refresh Token จาก TikTok API

4 เชื่อมต่อกับ NextGenBot

ใส่ App ID, Client Secret, Access Token และ Webhook URL ในหน้า Settings → เสร็จ!

💡 Tips: TikTok API ยังอยู่ในช่วงพัฒนา — ฟังก์ชันบางอย่างอาจต้องรอการอัปเดตจาก TikTok

Discord

เชื่อมต่อ Discord Bot เพื่อจัดการ Community, Ticket System และ Moderation

1 สร้างแอปบน Discord Developer Portal

ไปที่ https://discord.com/developers/applications → New Application → ตั้งชื่อ

2 สร้าง Bot Token

ไปที่ Bot → Add Bot → Reset Token → คัดลอก Token มาเก็บ

3 เชิญบอทเข้า Server

ไปที่ OAuth2 → URL Generator → เลือก scopes: bot → เลือก permissions → เปิด URL เพื่อเชิญบอท

4 เชื่อมต่อกับ NextGenBot

ใส่ Bot Token + Guild ID + Channel ID ใน NextGenBot Settings → Save → บอทเริ่มทำงาน!

💡 Tips: เปิด Intents: MESSAGE CONTENT INTENT, SERVER MEMBERS INTENT ใน Developer Portal ด้วย

Instagram

เชื่อมต่อ Instagram Business Account เพื่อตอบ DM และ Comment อัตโนมัติ

1 เปลี่ยนเป็น Business Account

ใน Instagram → Settings → Account → Switch to Professional Account → เลือก Business

2 เชื่อมต่อกับ Facebook Page

Instagram Business ต้องเชื่อมกับ Facebook Page — ตั้งค่าใน Meta Business Suite

3 สร้าง Access Token

ใน Meta for Developers → ใช้ Graph API เพื่อขอ Instagram Business User ID + Access Token ขอบเขต instagram_basic, instagram_manage_messages, instagram_manage_comments

4 เชื่อมต่อกับ NextGenBot

ใส่ Instagram Business ID + Access Token + Facebook Page ID ใน Settings → เสร็จ!

💡 Tips: Instagram API จำกัดการตอบ DM เฉพาะข้อความที่ลูกค้าส่งมาก่อน — ไม่สามารถเริ่มแชทก่อนได้

Website Chat Widget

ติดตั้ง Chat Widget บนเว็บไซต์ของคุณ เพียงแค่แปะโค้ดสั้นๆ

1 เปิดการใช้งาน Chat Widget

ใน NextGenBot Settings → Website Widget → เปิด toggle "Enable Chat Widget"

2 ปรับแต่งหน้าตา

เลือกสี ปุ่ม แชทบับเบิล ข้อความต้อนรับ และตำแหน่ง (ซ้าย/ขวา) ตามแบรนด์ของคุณ

3 คัดลอกโค้ด

กดปุ่ม Copy Code จะได้โค้ดแบบนี้:

<script>
  window.nextgenbotWidget = { appId: "YOUR_APP_ID" };
</script>
<script src="https://widget.nextgenbot.com/chat.js"></script>
4 แปะบนเว็บไซต์

นำโค้ดไปแปะก่อน </body> ในทุกหน้าของเว็บไซต์คุณ — เสร็จ! Widget จะแสดงโดยอัตโนมัติ

💡 Tips: ทดสอบ widget โดยเปิดหน้าเว็บแล้วคลิกแชทบับเบิล — ข้อความจะเข้ามาที่ Inbox ของ NextGenBot ทันที
